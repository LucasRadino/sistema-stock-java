# Sistema de Stock y Ventas

Aplicación desarrollada en Java con MySQL que permite gestionar productos y registrar ventas.

## Funcionalidades

- Crear productos
- Listar productos
- Registrar ventas (actualiza el stock automáticamente)
- Eliminar productos con validación de integridad

## Tecnologías

- Java
- MySQL
- JDBC

## Conceptos aplicados

- CRUD
- DAO (Data Access Object)
- Manejo de base de datos relacional
- Validación de lógica de negocio
- Integridad referencial

## Cómo ejecutar

1. Crear la base de datos usando el archivo `database.sql`
2. Configurar usuario y contraseña en `ConexionDB.java`
3. Ejecutar el proyecto desde Eclipse
